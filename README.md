Ez a scratch projekt a programozás nylevek 1 tantárgyra készült.

A projekt egy fogócska játékot tartalmaz, amelyben a batter sprite-al kell üldözni a macskát.A bal felső sarokban látható egy counter ami a macska életét jelzi ha ez a counter eléri a nullát akkor lejátszik egy hangot és véget ér a program.
